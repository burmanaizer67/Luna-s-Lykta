# Luna's Lykta / Luna's Lantern

En interaktiv saga om mod / An interactive tale of courage

## Om projektet / About the Project

Luna's Lykta är en interaktiv webbsaga för barn om Noah som får en magisk lykta från Luna, väktaren av nattens stigar. Sagan finns på både svenska och engelska.

Luna's Lantern is an interactive web story for children about Noah who receives a magical lantern from Luna, the guardian of the night's paths. The story is available in both Swedish and English.

## Språk / Languages

- **Svenska**: [index.html](index.html)
- **English**: [index-en.html](index-en.html)

## Hur man använder / How to Use

1. Öppna `index.html` för svenska versionen / Open `index.html` for the Swedish version
2. Öppna `index-en.html` för engelska versionen / Open `index-en.html` for the English version

Ingen installation krävs - öppna bara HTML-filerna direkt i din webbläsare!
No installation required - just open the HTML files directly in your web browser!

## GitHub Pages

För att publicera sidan live på GitHub Pages:
To publish the site live on GitHub Pages:

1. Gå till repository Settings / Go to repository Settings
2. Bläddra ner till "Pages" under Code and automation
3. Under "Source", välj "Deploy from a branch"
4. Under "Branch", välj "main" och "/root"
5. Klicka "Save"

Din sida kommer vara tillgänglig på: `https://[ditt-användarnamn].github.io/[repository-namn]`
Your site will be available at: `https://[your-username].github.io/[repository-name]`
